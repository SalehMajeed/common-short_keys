```bash
    ctrl + shift + t
```

# open new tab

```bash
    ctrl + t
```

# open incognito mode

```bash
    ctrl + shift + n
```

# open developer tools

```bash
    ctrl + shift + i
```

```bash
    ctrl + d
```

```bash
    ctrl + l
```
